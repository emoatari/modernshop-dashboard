# ModernShop - Admin Dashboard

**A complete e-commerce admin dashboard built with Bootstrap 5, Chart.js, and JavaScript**


---

## 📌 Project Overview

This is a fully responsive admin dashboard for e-commerce management. Built as a practice project to demonstrate front-end development skills.

**Key Features:**
- 📊 Sales statistics cards with dynamic data
- 📈 Interactive sales chart (Chart.js)
- 📋 Orders table with delete functionality
- 🏷️ Product tabs (All / Electronics / Accessories)
- 📱 Fully responsive with mobile sidebar toggle
- 🔔 Toast notifications for user actions
- 🗑️ Delete order with confirmation modal

---

## 🛠️ Technologies Used

| Technology | Usage |
|------------|-------|
| HTML5 | Structure |
| CSS3 | Custom styling |
| Bootstrap 5.3 | Layout, components, grid |
| Chart.js | Sales chart visualization |
| JavaScript | Interactivity (delete, modal, tabs, toasts) |
| Bootstrap Icons | Icon library |
